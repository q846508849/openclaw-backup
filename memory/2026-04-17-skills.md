# 2026-04-17 新增技能

## 浏览器自动化类

### Agent Browser (agent-browser-clawdbot)
- 功能：OpenClaw 浏览器自动化技能，通过 CDP 协议操控 Edge/Chrome
- 依赖：浏览器需开启 `--remote-debugging-port`（Edge:9222 / Chrome:9223）
- 无 API Key

### Playwright MCP (playwright-mcp)
- 功能：Playwright 自动化 + MCP 协议 + 爬虫
- 依赖：需安装 Playwright（`npx playwright install`）
- 无 API Key

### Browser Automation CDP (browser-automation-cdp)
- 功能：通过 CDP WebSocket 与浏览器通信，支持 goto/eval/click/screenshot
- 依赖：浏览器开启 remote debugging 端口
- 无 API Key
- 安全：VirusTotal 误报，代码已人工审核，无风险

### OpenClaw CLI (openclaw-cli)
- 功能：OpenClaw 生态管理技能（Gateway、Channels、Models、Automation、Nodes、Deployment）
- 依赖：OpenClaw CLI 环境
- 无 API Key

### 小红书搜索建议关键词收集工具 (xhs-suggest-keywords)
- 功能：从小红书搜索框获取搜索建议关键词，用于关键词收集和趋势分析
- 依赖：可能需要小红书登录态（通过浏览器 CDP）
- 无 API Key
